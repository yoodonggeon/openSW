# openSW
오픈소스SW기초 수업 과제

강의실 내 콘센트 위치를 사진을 통해 확인할 수 있는 프로그램

1. 단국대학교 홈페이지에 나와있는 소방 대피도 사진을 다운받는다.
2. 'leaflet'이라는 github에서 제공하는 프로그램을 활용하여 사진에 마커를 찍는다.
https://github.com/Leaflet/Leaflet
3. javascript, HTML, CSS를 활용하여 마커를 찍은 사진을 보여줄 수 있는 ui를 만든다.

목표 : 소프트웨어 516호 콘센트 위치를 파악할 수 있는 프로그램을 만든다.

외부 라이브러리(오픈소스) : leaflet.js를 목적에 맞게 사용
나중에 소화기 위치, 와이파이 공유기 위치 등을 추가하여 확장할 수 있는 가능성이 있다.
